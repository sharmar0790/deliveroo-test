# Deliveroo Test

### Requirements
  - java 11
  
### Steps to run the program

  -  Build the code to have the executable jar
    ```
    $ mvn clean package
    ```
  - Run the program
    ```
    $ java -jar deliveroo-test-1.0-SNAPSHOT.jar {String_to_be_tested}
    EX: java -jar deliveroo-test-1.0-SNAPSHOT.jar ＂*/15 0 1,15 * 1-5 /usr/bin/find＂
    ```
